About AWS Quiz Challenge

This project is a comprehensive AWS knowledge assessment tool built with modern web technologies.

Technologies Used:

  • HTML5, CSS3, and JavaScript
  • AWS Lambda for backend services
  • Amazon DynamoDB for data storage
  • Amazon API Gateway for RESTful APIs
  • Hosted website files on Amazon S3
  • Deployed using AWS Amplify
